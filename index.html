Bash/Shell Cheatsheet

Commands

Command Stands For Description
pwd Print Working Directory Displays the file path from the root directory to the current working directory.
ls list Used to list the contents of a directory. If no arguments are given, it will list the contents of the current working directory.
cd Change Directory Used to move throughout the filesystem of a computer. It accepts a variety of arguments:
mkdir Make Directory Used to make a new directory in the filesystem according to its argument.

Options

Option Description
-a show all. Forces ls to show fileand directory names beginnning with . cd on its own returns to home directory
-R list all nested subdirectories within a directory
-i (interactive) used in conjunction with rm command we have a chance to check that we are deleting only the files that we want to remove
-l long listing format (file size, last modification)
-r lists contents in reverse order
-h human readable
-F Tells ls to classify output by adding a marker to file and directory names to indicate what they are.
